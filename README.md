# Aspect-Based Sentiment Analysis for Product Reviews

## Overview
This academic project focuses on Aspect-Based Sentiment Analysis (ABSA) to analyze product reviews. Instead of predicting the overall sentiment of a review, the system identifies specific product aspects (e.g., battery, screen, price, and quality) and determines the sentiment associated with each aspect.

## Features
- Text preprocessing and data cleaning
- Sentiment analysis using VADER (NLTK)
- Aspect extraction using keyword matching
- Aspect-level sentiment classification
- Data visualization using tables and charts
- Performance evaluation using Accuracy, Precision, Recall, and F1-Score

## Dataset
The project uses product review datasets collected from e-commerce platforms for sentiment analysis and aspect extraction.

## Results
The model achieved approximately:
- Accuracy: 80%
- Precision: 64%
- Recall: 80%
- F1-Score: 71%
